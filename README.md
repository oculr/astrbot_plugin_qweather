# 基于和风天气的查询天气插件
fork自[astrbot_plugin_get_weather_cmd](https://github.com/whzcc/astrbot_plugin_get_weather_cmd)

仅支持和风天气的[JWT认证](https://dev.qweather.com/docs/configuration/authentication/#json-web-token)

[和风天气开发文档](https://dev.qweather.com/docs/api/)

[和风天气开发者控制台](https://console.qweather.com/home)

使用方法
----
`/天气 <城市> [时间]`

可使用的时间格式如下：
`now`, `3d`, `7d`, `10d`, `15d`, `30d`, `24h`, `72h`, `168h`
